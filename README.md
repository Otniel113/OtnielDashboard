# OtnielDashboard
Kumpulan dashboard visualisasi data dari berbagai macam tools. Salah satunya yaitu Bokeh, Python untuk interaktif.

## Deskripsi Proyek
Proyek ini berisi berbagai dashboard visualisasi data interaktif yang dibangun menggunakan Python (terutama library Bokeh) dan beberapa visualisasi web statis lainnya. Link demo dapat diakses di bawah ini.

## Daftar Dashboard
Beberapa dashboard yang tersedia dalam repositori ini:
- **BMKG Visualization**: Visualisasi data cuaca dari BMKG seperti suhu, angin, kelembapan, curah hujan, dan lain-lain.
- **National HDI**: Perbandingan Human Development Index (HDI) antar berbagai negara.
- **VN Producer Network**: Analisis jejaring sosial produser (publisher/developer) dalam industri Visual Novel.
- **Gempa Bumi**: Data gempa bumi terkini.
- **Top Anime 2020**: Visualisasi daftar anime terbaik pada tahun 2020 berdasarkan skor Myanimelist dengan race bar chart.

## Teknologi yang Digunakan
- **Python**: Bahasa pemrograman utama.
- **Bokeh**: Library untuk membuat visualisasi interaktif di web.
- **Pandas & NumPy**: Untuk pemrosesan dan analisis data.
- **HTML/CSS (MDBootstrap)**: Untuk antarmuka halaman utama (landing page).
- **SigmaJS**: Untuk visualisasi jaringan.
- **Flourish Studio**: Visualisasi interaktif no-code yang bisa di-embed ke HTML.
